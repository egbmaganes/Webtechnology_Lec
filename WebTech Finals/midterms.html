SELECTORS

Selectors are used to find certain parts of the html which can then be used for the CSS Rule.

CSS Basic Selector

1 or more sequences of simple selectors separated by combinators with 1 pseudo-element possibly appended to the last sequence

  • Universal Selector - allows you to select and style any element type.

    • Example: *{color:white}

  • Element Selector - allows you to select and style a HTML element.

    • Example: div{color:white}

  • Id Selector - allows you to select and style the element with the specified id.

    • Example: #Some_ID{color:white}

  • Class Selector - allows you to select and style the elements with the specified class.

    • Example: .Some_Class{color:white}

  • Element specified selector - allows you to select and style only the elements associated with a specified class/id.

    • Example: div.Some_Class{color:white}

Relational Selectors

  • Descendent selector - allows you to select and style all elements that are descendants of a specified element.

    • Example: div h3{color:white}

  • Child selector - allows you to select and style all elements that are direct descendants.

    • Example: div>h2{color:white}

  • Adjacent sibling selector - allows you to select and style the elements that is an immediate/ adjacent sibling.

    • Example: p+span{color:white}

  • General sibling selector - allows you to select and style the elements that are siblings no matter the arrangement.

    • Example: h1~h3{color:white}

Attribute Selectors

  • Attribute Selector - allows you to select and style an element with a specified attribute or attribute value

    • Example: input[type="text"]{color:white}

Pseudo-Classes

  • Pseudo-class selector - allows you to select and style an element with a special state specified by a keyword

    • Example: a:hover{color:white}

  • nth-child selector - allows you to select and style an element that has an+b-1 siblings before it in the doc tree and has a parent element

    • Example: a:nth-child(4n){color:white}

Javascript

Developed circa 1995 by Brendan Eich at Netscape Communications as the scripting language for the Netscape Navigator Browser. Formally called Mocha, after some time it became LiveScript, then finally JavaScript. The JavaScript is standardized by ECMA International as ECMAScript.

Versions

Latest Version

  • JavaScipt 1.8.1
  • ECMAScript 5 [ECMA-262 5th Edition]

Common Version

  • JavaScript 1.5
  • Jscript 5.5
  • ECMAScript v3 [ECMA-262 3rd Edition]

Three ways to call a script

  • Embedded Script - the codes of the script are within the html file
  • Externally Link Script - link a .js file from your html using a script markup

Loading of the JavaScript

  • defer - indicates that the browser executes the script once the document has been parsed.
  • async - indicates that the browser executes the script asynchronously if possible.

Because of DOM, Javascript can access and change the contents of the html document.

Example of Javascript and its return values:

  • h1.innerText = “hello”;
  • h1.style = “color:blue”;
  • h1.style.color = “red”;
  • h1.style.fontsize = ‘2em’;

Subsection of a Document

  • querySelectorAll(‘p:first-child’)

  • querySelectorAll(‘body h1 p’)
  • querySelectorAll(‘h1 + p’)

  • Functions accepts argument(any type)
  • Functions that can return any type of value
  • Functions can be recursive
  • Functions can be nested
  • Function arguments can have default values

Array - in JavaScript, index can be non-contiguous

Multicore Methods

  • .push - add 1 element from the end of the array
  • .shift – remove 1st element then move the elements from right to left by one
  • .reverse – reversed the position of the elements
  • .splice – add/removes items to/from an array then return the specified item


Methods

  • Mutators – modify arrays
    • Array.prototype.fill()
    • Array.prototype.pop()
  • Accessor – does not modify arrays
    • Array.prototype.concat()
    • Array.prototype.indexOf()
    • Array.prototype.join()
  • Iterators – execute something for each elements in array
    • Array.prototype.filter()
    • Array.prototype.map()
    • Array.prototype.every()
    • Array.prototype.reduce()
