Web Technologies

The world that we live in is changing dramatically over the period of time ever since man revolutionized the world by the invention of the modern technologies that we now use and encounter in our daily lives. One of such is called the internet. A term for the internet which best describes its purpose is the information superhighway. When used expertly, basically the internet gives you the power to shrink the world with the use of your computer to gain access to information and knowledge on almost every subject matter that you are curious about with only the use of the palm of your hands.

"The power of the Web is in its universality. Access by everyone regardless of disability is an essential aspect."

- Tim Berners-Lee

The Course Website

Our team created a course website that covers information about Web Technology. It provides information about the behaviour of the request-response protocol in the internet. The creation of the documents (web pages) in regards to structure, aesthetics, and behaviour that can be parsed and viewed by our browsers. Lastly, it provides information regarding server-side scripting that executes commands within the server and not on the client. We are also going to tackle some Web Application Security information along the end of this course.

Prelims

How are transactions made in the World Wide Web? What is the difference between the Internet and World Wide Web? Learn how requests are made in the World Wide Web, how servers treat those request via sending a response, and some status codes that goes with it. Click here to start Prelims.

Midterms

Discover and learn the basic fundamentals of HTML, CSS, and JavaScript(Client-side Scripting). Learn how the structure of HTML works, how designs are made in certain webpages, and the behavioral content of it. Click here to start Midterms.

Finals

Learn how Server-side Scripting are capable of producing dynamic contents. Discover some available programming languages which you can use as your web server. Lastly, learn also Web Application Security to go with your 'soon to be' web applications. Click here to start Finals.
